# SmartChat AI

A web application that enables real time AI based conversations using local language models with a full stack architecture:
1. **Backend System**: FastAPI based API for handling requests and chat processing  
2. **AI Integration**: Ollama powered models such as Gemma for generating responses  

## Features

- Real time AI chat interaction  
- Persistent chat history using PostgreSQL  
- Support for multiple AI models  
- Fast and scalable backend using FastAPI  
- Modern and responsive web interface

Running the Application
Quick Start
Start Ollama:
ollama serve
ollama pull gemma3:4b
Run the backend:
cd backend
python -m uvicorn main:app --reload
Run the frontend:
cd frontend
npm run dev
Open your browser:
http://localhost:5173
First Run

On the first run, the application will:

Initialize database tables
Connect to PostgreSQL
Load available AI models
Start backend and frontend servers

Note: Initial setup may take a few seconds depending on system performance.

What You'll See
Chat Interface: Clean UI for real time conversations
AI Responses: Instant responses generated by language models
Chat History: Stored conversations for future access
Troubleshooting
Port already in use? Change backend or frontend port
Missing dependencies? Run pip install -r requirements.txt
Ollama not working? Ensure it is running using ollama serve
Database issues? Check PostgreSQL service status
